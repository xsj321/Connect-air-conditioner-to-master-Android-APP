# Connect-air-conditioner-to-master-Android-APP
APP配套基于STM32F107的互联空调主控 链接见描述

使用SDK23配套[STM32F107互联空调主控](https://github.com/xsj321/Main-control-of-interconnected-air-conditioner-based-on-STM32F107)

## 主要功能：


1. 实现对主控消息的同步;
   - 同步主控采集到的温湿度；
   - 同步主控设定的温度；
   
2. 本地化账号密码数据库；
   - 登陆和注册账号密码；
   - 自动登陆；

3. 调节目标温度；
